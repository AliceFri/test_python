JavaScript 的数据类型，共有六种

    1. 数值: 整数 和 小数

    2. 字符串
    
    3. 布尔值

    4. undefined: 未定义 或 不存在

    5. null: 空值

    6. 对象： 各种值组成的集合

数值、字符串、布尔值这三种类型，合称为原始类型（primitive type）

对象是最复杂的数据类型，又可以分成三个子类型。

    1. 狭义的对象 object
    
    2. 数组 array

    3. 函数 function

typeof: 返回一个值的数据类型

instanceof: 是否为其实例
